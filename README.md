# Robot SLAM Lab

An interactive mapping simulation showing how a robot can discover obstacles and build an occupancy map over time.

## Highlights
- Simulated lidar rays with configurable scan density.
- Autonomous movement with collision-aware steering.
- Side-by-side visualization: ground-truth world vs discovered map.
- Live metrics for coverage, obstacle discovery, and trajectory length.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS controls

## Run
Open `index.html` in your browser.
